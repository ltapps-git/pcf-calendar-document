# LTAPPS Calendar PCF Control

LTAPPS Calendar is a powerful and flexible PowerApps Component Framework (PCF) control designed to enhance scheduling, planning, and event visualization across Microsoft Power Apps.

Built to support Model-Driven Apps, Canvas Apps, and Custom Pages, the control delivers a modern, customizable, and highly interactive calendar experience. Whether you're managing projects, scheduling resources, or tracking company events, LTAPPS Calendar provides the tools you need for effective time management and visualization.

## Key Features

- **Multi-Platform Compatibility**: Seamlessly integrates with Model-Driven Apps, Canvas Apps, and Custom Pages
- **Flexible Data Connectivity**: Connects to various data sources including Dataverse, SharePoint Lists, and other Power Platform connectors
- **Multiple Calendar Views**: Visualize events in Quarter, Month, Week, or Day views for improved clarity and navigation
- **Advanced Refiners & Filtering**: Supports multi-refiners allowing users to filter events based on multiple criteria with dynamic filtering capabilities
- **Recurrence Support**: Fully handles recurring events for more accurate scheduling
- **Customizable Callouts & Highlights**: Displays detailed event information through customizable callouts and highlights events based on configurable conditions
- **Custom Button Callbacks**: Enables developers to create and attach custom button actions for seamless integration into complex processes
- **Modern User Interface**: Features a clean, modern, and responsive UI designed for optimal user experience across devices

## Use Cases

- Project and resource scheduling
- Company event calendars
- Leave, shift, or booking management
- CRM activity and appointment overview

## Summary

LTAPPS Calendar PCF is a versatile and highly configurable solution for organizations seeking a robust and dynamic calendar component within the Power Platform. With its wide app support, flexible customization options, and modern design, it empowers makers and developers to build rich, user-friendly calendar experiences tailored to any business need.
